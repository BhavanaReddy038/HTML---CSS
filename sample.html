INTRO to HTML

        HTML (Hypertext Markup Language) is the standard markup language for creating web pages and web applications.

    !DOCTYPE html: Declaration defines that this document is an HTML5 document

    html tag: Root element of the webpage

    head tag: Contains metadata, links, CSS links, Scripts

    title tag: Specifies a title for the html page, 
            which is shown in the browser's title bar or in the page's tab

    body tag: Contains the content of the webpage, such as text, images, links, etc.

    h1 tag: Heading 1, used for main headings

    p tag: defines a paragraph of text

Headings & paragraph in Html & Basics

About headings: consists 6 headings in HTML. Each heading is defined by a h1 to h6(defines the least important heading) tag. Paragraphs are defined by the p element. If we break the lines in paragraph tags they won't change in web page. The browser will automatically remove any extra spaces and lines when the page is displayed: HTML tags are not case sensitive 

          Headings Are Important: Search engines use the headings to index the structure and content of your web pages.
          Users often skim a page by its headings. It is important to use headings to show the document structure.
          h1: headings should be used for main headings, followed by h2 headings, then the less important h3, and so on.

          For example:
          h1 - Page title
          h2 - Section titles
          h3 - Sub-sections

          Each HTML heading has a default size. However, you can specify the size for any heading with the style attribute, using the CSS font-size property:

          Tip: Use only one h1 per page - it represents the main topic or title.

          Note: Use HTML headings for headings only. Don't use headings to make text BIG or bold.

            HTML Links - The target Attribute
            By default, the linked page will be displayed in the current browser window. To change this, you must specify another target for the link.
            The target attribute specifies where to open the linked document.
            The target attribute can have one of the following values:
                _self - Default. Opens the document in the same window/tab as it was clicked
                _blank - Opens the document in a new window or tab
                _parent - Opens the document in the parent frame
                _top - Opens the document in the full body of the window

HTML Block & Inline Elements

            Block-level elements: Start on a new line and take up the full width available. 
            Examples: div, p, h1, ul, ol, li, table, etc.

            Inline elements: Do not start on a new line and only take up as much width as necessary. 
            Examples: span, a, strong, em, img, etc.

            Note: Some elements can be both block-level and inline, depending on their context and usage.
             An inline element cannot contain a block-level element!

HTML Semantic Elements

            Semantic elements: clearly describe their meaning in a human- and machine-readable way.
            Examples: header, footer, article, section, nav, aside, main, figure, figcaption, time, mark, etc.

            Non-semantic elements: do not clearly describe their meaning. 
            Examples: div, span, b, i, etc.

            header tag: Defines a header for a document or section
            footer tag: Defines a footer for a document or section
            article tag: Defines an article
            section tag: Defines a section in a document
            nav tag: Defines navigation links
            aside tag: Defines content aside from the page content
            main tag: Specifies the main content of a document
